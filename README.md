## 📝 Descripción General

Este repositorio contiene todos los laboratorios y actividades prácticas desarrolladas durante el semestre en el curso de **Minería de Datos**. El objetivo es aplicar técnicas de preprocesamiento, análisis exploratorio y modelado de datos para extraer conocimiento útil de conjuntos de datos reales.

## 📁 Estructura del Portafolio

El portafolio está organizado por semanas, reflejando el progreso gradual desde los fundamentos hasta técnicas avanzadas:

| Carpeta | Temas Principales |
| :--- | :--- |
| **Semana 1 - 2** | Introducción a Python para ciencia de datos, manejo de archivos CSV, primeros pasos con Pandas |
| **Semana 3 - 4** | Limpieza y preprocesamiento: valores nulos, datos atípicos, normalización y codificación |
| **Semana 5 - 7** | Análisis Exploratorio de Datos (EDA): visualizaciones, correlaciones, estadística descriptiva |
| **Semana 8 - 10** | Modelado supervisado: regresión lineal, clasificación (KNN, Árboles de Decisión) |
| **Semana 11 - 12** | Modelado no supervisado: clustering (K-Means), reducción de dimensionalidad (PCA) |

## 🚀 Actividades Más Relevantes para mi Aprendizaje

1.  **Limpieza y preparación de datos (Semanas 3-4):**  
    Aprendí que esta etapa consume más del 70% del tiempo en un proyecto real. Manejar valores nulos, outliers y datos inconsistentes fue un desafío inicial que ahora considero fundamental.

2.  **Implementación de modelos de clasificación (Semana 8-10):**  
    Pasar de la teoría a la práctica con algoritmos como KNN y árboles de decisión me permitió entender cómo se "entrena" una máquina para reconocer patrones. Comparar su precisión fue muy enriquecedor.

3.  **Aplicación de clustering (Semana 11-12):**  
    Utilizar K-Means para segmentar datos sin etiquetas y visualizar los grupos con PCA me pareció fascinante. Me ayudó a comprender el poder del aprendizaje no supervisado para descubrir estructuras ocultas.
